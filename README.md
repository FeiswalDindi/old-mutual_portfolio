## 🏗️ Architecture & Stack
This project was built with a focus on performance, semantic structure, and native browser capabilities without the overhead of heavy frameworks:
* **HTML5:** Semantic, accessible structuring.
* **CSS3:** Custom variables, CSS Grid/Flexbox, and glassmorphic UI elements.
* **Vanilla JavaScript:** Intersection Observers for high-performance scroll reveals and DOM manipulation.
